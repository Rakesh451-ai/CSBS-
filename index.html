Sort an Array in JavaScript

This example sorts an array of numbers in ascending order using Array.prototype.sort with a numeric comparator. 

const numbers = [5, 2, 9, 1, 7];
const sortedNumbers = [...numbers].sort((a, b) => a - b);

console.log(sortedNumbers); // [1, 2, 5, 7, 9]
